# GitHub Pages Сайт для Habr InfoSec RSS Бота

Добро пожаловать на GitHub Pages сайт для Habr InfoSec RSS Telegram бота! Этот сайт предоставляет веб-интерфейс для бота, который позволяет получать последние статьи по информационной безопасности с Хабра.

## Функциональность

Наш GitHub Pages сайт включает:

- **Интерактивный чат**: Симуляция работы Telegram-бота с настоящей функциональностью получения статей
- **Настоящая RSS интеграция**: Сайт напрямую подключается к RSS-ленте Хабра по информационной безопасности
- **Реальные статьи**: По командам `/infosec` и `/security` отображаются настоящие последние 10 статей с Хабра
- **Поддержка всех команд**: `/start`, `/help`, `/infosec`, `/security`

## Как использовать

1. Введите команду в поле ввода (например, `/infosec`)
2. Нажмите "Отправить" или Enter
3. Получите ответ от бота:
   - Для `/infosec` или `/security`: список из 10 последних статей по информационной безопасности
   - Для `/start` или `/help`: информационные сообщения о боте и его командах

## Технические детали

- Сайт использует библиотеку [feednami](https://github.com/richardgirges/feednami) для парсинга RSS-лент в браузере
- Подключается к официальной RSS-ленте Хабра: `https://habr.com/ru/rss/hub/infosecurity/all/?fl=ru`
- Статьи отображаются с заголовком, кратким описанием и ссылкой на оригинал
- Описание статей ограничено 200 символами для лучшего отображения

## Ограничения

- Из-за CORS политик некоторых RSS-источников функциональность может быть ограничена в некоторых браузерах
- В случае ошибок загрузки статей отображается сообщение об ошибке

## Telegram бот

Для полноценного использования рекомендуется использовать оригинального Telegram-бота:

1. Найдите бота в Telegram
2. Отправьте команду `/start`
3. Используйте команду `/infosec` или `/security`, чтобы получить последние статьи по информационной безопасности
